# Slot Tools

パチスロ機種別の立ち回り支援ツール集。

## 公開URL

**https://uma0123.github.io/slot-tools/**

iPhone Safariで開き、共有 → 「ホーム画面に追加」でPWAライクに使えます。

## 収録ツール

### 北斗の拳 転生の章2（スマスロ）
- **シャッター狙い記録ツール** — `hokuto-tensei2/shutter.html`
  周辺台を目視チェックする際の記録ツール。
  - 店舗・台レンジ管理
  - 開始条件（AT後/設変）× 現在あべし × シャッター（あり/なし/未確認）から期待出玉率を自動算出
  - 期待値順ソートで優先台が一目で分かる
  - AT後即やめ / 2周目以降続行（伝承モード考慮）の計算モード切替
  - データはブラウザのLocalStorageに保存（端末ごと）
