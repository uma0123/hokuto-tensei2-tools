# 北斗の拳 転生の章2 ツール集

スマスロ「L北斗の拳 転生の章2」(6.5号機 / サミー) 用の立ち回り支援ツールまとめ。

## 公開URL

**https://uma0123.github.io/hokuto-tensei2-tools/**

iPhone Safariで開き、共有 → 「ホーム画面に追加」でアプリのように使えます。

## 収録ツール

### 📊 シャッター狙い記録ツール
- **URL**: https://uma0123.github.io/hokuto-tensei2-tools/shutter/
- **パス**: `shutter/index.html`

周辺台を目視で監視する際の記録ツール。

- 店舗・台レンジ管理（複数店舗を切替）
- 開始条件（AT後 / 設変）× 現在あべし × シャッター(○/✕/?) から期待出玉率を自動算出
- 期待値順ソートで優先台が一目で分かる
- **AT後即やめ / 2周目以降続行**（伝承モード・持越しあべし考慮）の計算モード切替
- 期待値テーブル：すろらぼ・だくお氏シミュレートデータをベース
  - AT後／設変 × シャッター(狙い/有り/無し) × あべし帯別の完全テーブル
  - 384フェイク天命特例対応
- データはブラウザのLocalStorageに保存（端末ごと独立）

### 📝 北斗転生トラッカー（旧バージョン・候補管理）
- **URL**: https://uma0123.github.io/hokuto-tensei2-tools/tracker/
- **パス**: `tracker/index.html`

候補台に「シャッター来た / 来ない / 未確認」を付けて記録するシンプルな管理ツール。シャッター狙い記録ツールの前身。
